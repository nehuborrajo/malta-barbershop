# Malta BarberShop - App de Gestión de Turnos

Aplicación web para gestionar turnos de barbería.

## Funcionalidades
- Agenda diaria/semanal con vista de grilla
- Turnos fijos semanales
- Registro de pagos (Efectivo, Transferencia, Mercado Pago)
- Métricas e ingresos

## Ejecutar localmente
```bash
pip install -r requirements.txt
python app.py
```

Abrir http://localhost:5000
